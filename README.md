AI-Powered Directory Management System

Project Overview

The AI-Powered Directory Management System is a web-based application that helps users analyze, organize, and clean directories efficiently. It automatically categorizes files, identifies duplicates, and provides actionable insights on storage usage.

This project combines Python Flask backend for directory analysis and Vue.js frontend for a responsive, interactive interface with smooth UI/UX using Tailwind CSS.
